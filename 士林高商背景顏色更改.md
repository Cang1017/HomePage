### :computer: Enter this in Console:
```javascript=1
var body=document.querySelector("body");
var isBlue=false;
setInterval(function(){
if(isBlue)
{
    body.style.backgroundColor="#22313A";
}
else
{
    body.style.backgroundColor="#22313A";
}
isBlue = !isBlue;
},1000);
```
